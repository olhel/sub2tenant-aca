Azure Subscription ID to Tenant Lookup

Map any Azure Subscription ID directly to its Microsoft Entra tenant. sub2tenant performs an unauthenticated Azure Resource Manager request to extract the tenant from the WWW-Authenticate header and then uses Microsoft Graph to return the tenantId, displayName and defaultDomainName.

This is useful when a subscription ID appears in tickets, alerts, invoices or reports and tenant context is missing. No sign-in is required, and no subscription IDs or results are logged or stored.
