# Contributing to sub2tenant

Thanks for considering contributing!

## How to contribute

### 1. Fork the repository

Click **Fork** at the top-right of the GitHub page.

### 2. Create a feature branch

    git checkout -b feature/my-improvement

### 3. Make your changes

-   Keep code simple and readable\
-   Avoid adding dependencies unless necessary\
-   Ensure no secrets, tokens, or sensitive data are added

### 4. Test locally

    npm install
    npm start

### 5. Submit a Pull Request

Describe: - What you changed - Why it improves the project

## Reporting issues

Please include: - Steps to reproduce - The expected result - The actual
result - Screenshot or error snippet if relevant

## Code style guidelines

-   Use consistent formatting
-   Prefer small, focused commits
-   Keep backend logic minimal and secure

Thanks for helping improve the project!
